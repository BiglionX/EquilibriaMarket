开源 · Open Source

EquilibriaMarket 是一个多方注意力市场调度引擎， 实现 Vickrey 第二价格拍卖 + 可插拔治理插件 + 基尼系数监控。 理论基础来自 Michael I. Jordan（2018 NeurIPS Keynote） 关于"AI 与经济学融合"的研究方向。

本页是 Demo 模式（前端 mock 数据）。 真实部署请使用 cmd/admin-console（Go + Gin 后端）。
